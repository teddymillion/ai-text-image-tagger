  TROUBLESHOOTING

Problem: "API key not found"
Solution: Ensure .env.local exists in project root with the key

Problem: API calls failing
Solution: Verify free tier quota at https://console.cloud.google.com

Problem: Slow responses
Solution: First request loads model (~3s), subsequent requests are faster

Problem: Build fails
Solution: Run "pnpm install" first to get all dependencies
