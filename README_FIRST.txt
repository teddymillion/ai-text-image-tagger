                        ❓ QUICK TROUBLESHOOTING
================================================================================

Problem: "OPENAI_API_KEY is not set"
Solution: Make sure .env.local file exists and contains your API key

Problem: "Tags not generating"
Solution: Verify your API key is valid and your account has credits

Problem: "Text exceeds maximum length"
Solution: Reduce input text to under 5000 characters

Problem: "Build failed on Vercel"
Solution: Check Vercel logs - usually just needs environment variables

For more help: See README.md troubleshooting section
